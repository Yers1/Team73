# MedPrice.kz — презентация (7 слайдов)

> Аутлайн для защиты + сценарий живого демо. Каждый слайд — что показать и что сказать.

---

## Слайд 1 — Проблема
**«Сколько стоит сдать анализ крови в вашем городе?»**
- Пациент обходит десятки сайтов клиник вручную. Цены нигде не агрегированы, рынок непрозрачен.
- Разница за одну и ту же услугу между клиниками — **в 2–3 раза** (наши данные: «Холестерин» 800 → 2 720 ₸, +240%).
- 🎤 *«Для авиабилетов есть Aviasales. Для медицины — ничего. Мы это исправили.»*

## Слайд 2 — Решение
**MedPrice.kz — агрегатор и сравнение цен на медуслуги Казахстана.**
- Один поиск → список клиник с ценами, сортировкой и фильтрами.
- Автособираемые данные из открытых прайсов (PDF, DOCX, Excel, HTML).
- 🎤 Показать главный экран: поиск «общий анализ крови» → выдача по клиникам.

## Слайд 3 — Архитектура (техническая реализация)
- Pluggable-парсеры → **raw-слой** → **нормализация** → SQLite → FastAPI → React.
- Разделение raw/normalized, дедуп, журнал ошибок, очередь разметки — всё по ТЗ.
- Добавить источник = добавить адаптер, ядро не меняется.
- 🎤 Показать диаграмму из README + админ-панель (журнал парсинга 10 источников).

## Слайд 4 — Качество данных (наш ров)
- **8 клиник, 5 городов, 14 832 сырых строки → 2 050 предложений, 778 услуг.**
- Нормализация каскадом: **код тарификатора (3 558 совпадений, conf 1.0)** → синонимы → fuzzy (откалиброван).
- Чистим OCR-сканы: «10 80С» → 10 800, «ВОЗ.335» → код «B03.335».
- 🎤 *«Большинство команд застрянут на получении данных. У нас распарсены все 8 прайсов, включая сканы.»*

## Слайд 5 — Фичи, которые выигрывают
1. **🤖 AI-ассистент (чат-бот)** — понимает запрос своими словами и симптомы
   («болит горло», «проверить щитовидку»), подбирает анализы, находит цены и
   собирает корзину. На наших реальных данных, без галлюцинаций.
2. **🧺 Медкорзина** — оптимизатор: дешёвая раскладка по клиникам + «всё в одном месте». *Демо: чек-ап, экономия 68%.*
3. **📈 Реальная история цен** 2024→2026 по единым кодам (не синтетика).
4. **⚖️ «Честная цена»** — медиана рынка, бейджи «ниже/выше рынка».
- 🎤 Это закрывает «доп.функции» (15%) и добивает UX (25%).

## Слайд 7 — План развития и рынок
- **Источники:** live-парсинг kdl.kz, invitro.kz, helix.kz по расписанию (cron + APScheduler уже в архитектуре).
- **Города:** масштабирование на все областные центры РК.
- **Фичи:** подписка на снижение цены, онлайн-запись, Telegram-бот, маршрут 2GIS.
- **Монетизация:** лиды клиникам, продвижение, B2B-аналитика цен рынка.
- 🎤 *«Архитектура уже готова к десяткам источников. Сегодня — 8 клиник, завтра — весь Казахстан.»*

## 🎬 Сценарий живого демо (3 минуты)

1. **Главная.** «Это агрегатор цен на медуслуги. 2050 цен, 8 клиник, 5 городов — реальные данные из прайсов.»
2. **Поиск** → ввести «общий анализ крови» → автодополнение → выдача.
3. **Сравнение** (открыть услугу): «Одна услуга, 7 клиник. Дешевле всего — 800 ₸, дороже — 2720 ₸. Бейджи показывают, кто ниже/выше рынка. Зелёным — лучшая цена.»
4. **Честная цена + история:** показать медиану рынка и **динамику цен 2024→2026** (кнопка «динамика цен»).
5. **Код тарификатора:** «Метка „по коду тарификатора" — мы сопоставили по официальному коду, не по тексту. 3558 таких точных совпадений.»
6. **🧺 Медкорзина** (звезда демо): «Соберём базовый чек-ап.» Открыть пресет → «Система нашла дешёвую раскладку: ОАК в одной клинике, глюкоза — там же, приём — в другой. **Итого 6680 ₸ вместо 18 000 — экономия 68%.** Или всё в одном месте за 8800 ₸.»
7. **🤖 AI-ассистент** (нажать кнопку справа внизу): спросить «проверить щитовидку» или «где дешевле сдать кровь на сахар» → «Бот понимает обычный язык, подбирает анализы и сам собирает корзину — нажимаем „Открыть Медкорзину".»
8. **Карта:** клиники по городам.
9. **Админка:** «Журнал парсинга всех источников, raw-слой, очередь ручной разметки — всё по ТЗ.»
9. 🎤 Финал: *«Прозрачные цены экономят пациенту деньги и время. И это работает на реальных данных уже сейчас.»*
