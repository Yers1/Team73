# PRODUCT.md — MedPrice.kz

**Что это.** Агрегатор и сравнение цен на медицинские услуги Казахстана —
«Aviasales для медицины». Пациент вводит услугу (или симптом) и видит, в какой
клинике пройти её дешевле, с прозрачной ценой и историей.

**Кто пользователь.** Обычный человек (пациент), который хочет сэкономить и не
обходить десятки сайтов клиник. Не врач, не специалист — интерфейс должен быть
понятен с первого взгляда, вызывать доверие (это про здоровье и деньги).

**Register: product.** Это инструмент: пользователь решает задачу (найти, сравнить,
собрать корзину). Дизайн служит задаче, а не наоборот. Эталон доверия — Stripe,
Linear, уровень клинической точности. Никакого декора ради декора.

**Ключевые поверхности.**
- Главная — вход, поиск, готовые чек-апы, популярные сравнения.
- Выдача поиска — список услуг с фильтрами.
- Сравнение услуги — цены по клиникам, «честная цена» vs медианы, история.
- Медкорзина — оптимизатор стоимости набора услуг.
- AI-ассистент — чат: запрос/симптом → анализы и цены.
- Карта клиник, Админ-панель (журнал парсинга, очередь разметки).

**Тон.** Спокойный, точный, заслуживающий доверия. Уверенность через ясность и
пространство, не через украшения. Цвет — сигнал (действие, выгода, статус), не фон.

**Анти-слоп.** Без декоративных эмодзи. Без шумных градиентов. Единая система
line-иконок. Движение — только статусное (150–220 мс). Familiarity > сюрприз.
