DELIVERABLES — MedPrice.kz (кейс MedServicePrice), 2026-06-28
==========================================================

СОДЕРЖИМОЕ:

1. medservice.db
   База данных SQLite со всеми спарсенными и нормализованными данными.
   Открывается любым SQLite-клиентом (DB Browser for SQLite — бесплатно).
   Таблицы:
     clinics            — клиники (название, город, адрес, телефон, режим, рейтинг)
     sources            — журнал парсинга источников (файл, формат, год, статус, строк)
     raw_prices         — СЫРОЙ слой: строки как в источнике (аудит)
     services           — СПРАВОЧНИК услуг (id, специальность, название, категория,
                          код тарификатора, синонимы)
     normalized_prices  — нормализованные цены (привязка к справочнику, метод, confidence)
     offers             — витрина: актуальная цена по паре «клиника × услуга»
     price_history      — история цен по годам
     unmatched          — очередь ручной разметки (непривязанные услуги + AI-подсказка)

2. справочник_услуг.csv / справочник_услуг.xlsx
   Справочник услуг отдельным файлом: ID, Специальность, Код, Наименование,
   Категория, Код тарификатора, Синонимы. Всего позиций: 1230.
   (CSV в кодировке UTF-8 с BOM — корректно открывается в Excel.)

3. цены.csv / цены.xlsx
   Все нормализованные цены: Клиника, Город, Услуга, Категория, Цена (тг), Год,
   Код тарификатора, Источник. Всего записей: 2050.

4. medservice_dump.sql
   Полный SQL-дамп базы (на случай, если нужно восстановить БД из текста).
